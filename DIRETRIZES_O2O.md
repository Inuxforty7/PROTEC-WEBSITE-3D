# DIRETRIZES ESTRATÉGICAS PARA DESENVOLVIMENTO DE LANDING PAGE (O2O)

**Objetivo do Projeto:** Criar uma Web App / Landing Page focada em Alta Conversão para negócios locais (O2O - Online to Offline). O site vai receber tráfego de anúncios (Ads) e o objetivo único é gerar leads para o WhatsApp, reter clientes em grupos/comunidades e direcionar tráfego para a loja física.

## Regras de Estrutura e Funções Obrigatórias:

### 1. Posicionamento Premium (Hero Section):
* O design deve transmitir autoridade imediata no nicho.
* A promessa principal (O que fazemos) deve estar clara nos primeiros 3 segundos, acima da dobra.
* O botão de Call to Action (CTA) principal deve ter destaque visual absoluto (ex: animação shimmer/glow) e levar diretamente ao WhatsApp.

### 2. Elementos de Redução de Risco:
* Perto dos botões principais, adicionar "badges" (etiquetas) que quebrem objeções. Exemplo: "Orçamento Grátis", "Diagnóstico Rápido", "Resposta Imediata".
* Usar indicadores visuais (como um ponto verde a piscar) para simular que a equipa está "Online agora".

### 3. Prova Social Dinâmica:
* Implementar uma secção de Prova Social logo após a dobra inicial (preferencialmente uma faixa de animação infinita / Marquee, otimizada para mobile).
* Deve conter: Avaliações (ex: 5 Estrelas no Google) e Números de Autoridade (ex: +1000 clientes satisfeitos).

### 4. Funil de Contacto Estratégico (Rodapé/Fim da Página):
A secção de contacto não deve ter apenas um número, deve resolver 3 intenções do utilizador:
* **Ação Imediata:** Botão de WhatsApp Direto (Para quem quer comprar/agendar agora).
* **Retenção/LTV:** Botão para Grupo de WhatsApp, Comunidade ou Catálogo (Para quem está a pesquisar e queremos manter no nosso funil para o futuro).
* **Tráfego Físico:** Botão com link direto para o Google Maps (Para facilitar a visita à loja).

### 5. Otimização Mobile-First:
* Todos os botões devem ter área de toque grande.
* Textos e provas sociais devem empilhar corretamente em ecrãs pequenos.
* Nenhuma animação deve quebrar o layout no telemóvel.
